Using Mass Notification Documentation

This section:

  • Outlines the structure and contents of this guide
  • Provides a list of conventions used in the Mass Notification documentation set
  • Identifies other related documentation

Contents of This Guide

This guide contains the following chapters and appendixes.

Chapter or Appendix Description
Chapter 1, Introduction Introduces you to the Crisis Management solution from Everbridge. 
Chapter 2, Single Event Dashboard Describes the Critical Event Details page and the reports you can generate.
Chapter 3, Manage a Live Critical Event Describes how to create a dashboard for one critical event.
Chapter 4, Impact Tracker Introduces Impact Tracker for an event. It leverages the form a user has created and launches it to collect real-time event impact to the company’s assets, business, employees, and plans.
Chapter 5, Critical Event Templates Provides the procedures to create required templates for your critical event.
Chapter 6, Manage Task List Templates Provides instructions to create task list templates for your critical event.
Chapter 7, Document Library Provides details to maintain Documents in your critical events.
Chapter 8, Form Library Provides details to create and use forms in your critical events.
Chapter 10, Critical Events in the Everbridge Mobile App Provides details to use critical events in the Everbridge Mobile App.

Related Documentation

Documentation is provided to help you implement and run the Mass Notification of products. In addition to this publication, the Everbridge Suite documentation set consists of the following guides. You can view and/or download the PDF-formatted guides from Everbridge University. Print the PDFs in duplex (2-sided) mode to save resources.

Description

Intended Audience

Everbridge Mobile App User Guide

The Everbridge Mobile App is available for Apple iPhone, Apple iPad, Android Smart Phones, and tablets. It enables two-way communications with your members. They can reply to surveys and supply additional information, including pictures and their location.

Administrators, contacts, and members who use the Everbridge Mobile App

Everbridge Open User Guide

The Everbridge Open User Guide helps you learn about the two-way capabilities that allow third-party systems to create powerful closed-loop integrations with our Critical Event Platform.

Users who will manage Everbridge Open connectors: Safety connectors, ITA connectors, and Risk Event connectors

The reference source for using the production system. It provides information about the Mass Notification product to enable you to quickly set up configuration options and then use the system.

Administrators and operators

Account Administrators, Incident Administrators, and Incident Operators (users who will manage incidents and/or manage the communication plan for the organization)
