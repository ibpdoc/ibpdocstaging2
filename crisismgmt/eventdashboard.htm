Single Event Dashboard

The single event dashboard gives you a high-level view of what's happening in the critical event with a link you can share with others. By default, each event has a basic dashboard with the most common widgets. You can customize that dashboard to include for the particular event or even create different dashboards for different stakeholders.

Depending on a user's permission, they can also use the dashboard to edit event details and associated communications, task lists, and documents. You can also add notes to the event and comments on the dashboard.

Add a new single event dashboard

If you only need one dashboard, you can also edit the default dashboard to suit your needs—see Edit a single event dashboard.

  1. Open the critical event you want to edit the dashboard for (for example, clicking on the event in the Events dashboard or in the Events list view), make sure you're on the event Home tab then click Configure > Create New.
  2. Enter a name for the dashboard.
  3. Select event, task list, or custom widgets you want to include in the dashboard.
    1. You can also create a custom widget from the Add a Dashboard dialog box.
      • From the Widget Type field, select Custom Text or Webpage from the menu.
  4. Click Next .
  5. Select who you want to share the dashboard with:
    • All Users: Share with all users within the organization. This does only includes users who have access to the manager portal, not external contacts (you can share with contacts in the next step).
    • Private : only organization administrators and invited users can see the dashboard. If you select Private, add the roles or users you want to have access to the dashboard.
  6. Click Next to share with selected contacts.
  7. Click any of the squares to open the Select Contacts dialog where you can specify the contacts you want to share with by selecting individuals, groups, or rules.
  8. Click Select after you're finished selecting contacts to returned to the Add Dashboard steps.
    • Optionally, click Preview Contacts to see the list of contacts from each category.
    • Optionally, click Clear to remove all contacts from the list and start over.
  9. Click Save. The new dashboard appears on the event Home tab beside the existing ones.

Everbridge sends a notification to all selected contacts with a link to the new dashboard. By default, this is sent via mobile push, email, and SMS but this also depends on the setting in the event template. See Viewing Single Event Dashboards from the Member Portal.

Edit a single event dashboard

  1. Open the critical event you want to edit the dashboard for (for example, clicking on the event in the Events dashboard or in the Events list view) and make sure you're on the Home tab.
  2. Click Manage Dashboard on the dashboard you want to edit.
  3. Select the Event Widgets, Task Widgets, and Individual Task Lists, if available, that you want to see on your dashboard or clear those you want to remove.
    1. You can also create a custom widget from the Add a Dashboard dialog box.
      • From the Widget Type field, select Custom Text or Webpage from the menu.
  4. Click Share Internally to change the users in your organization that the dashboard is shared with:
    • All Users: Share with all users within the organization. This does only includes users who have access to the manager portal, not external contacts (you can share with contacts in the next step).
    • Private : only organization administrators and invited users can see the dashboard. If you select Private, add the roles or users you want to have access to the dashboard.
  5. Click Share Externally to change to contacts the dashboard is shared with.
  6. Click any of the squares to open the Select Contacts dialog where you can specify the contacts you want to share with by selecting individuals, groups, or rules.
  7. Click Select after you're finished selecting contacts to returned to the Manage Dashboard dialog box.
    • Optionally, click Preview Contacts to see the list of contacts from each category.
    • Optionally, click Clear to remove all contacts from the list and start over.
  8. Click Save when you're finished editing the dashboard.
  9. See Single event dashboard widgets for information on configuring the widgets.

Everbridge sends a notification to all selected contacts with a link to the new dashboard. By default, this is sent via mobile push, email, and SMS but this also depends on the setting in the event template. See Viewing Single Event Dashboards from the Member Portal.

Resizing the Dashboard Panes

To resize the Dashboard panes:

  1. Move the panes on the Dashboard by selecting the title and dragging the pane to the desired location.
  2. Hover the mouse in the lower right-hand corner of a pane and drag it to the size you want.

Single event dashboard widgets

You can select from a number pre-built widgets for common use cases or add custom widgets tailored for your organization and a particular event type.

Editing widget title

To edit the title of a widget, click the menu icon () and select Edit.

Removing a widget

To edit the title of a widget, click the menu icon () and select Remove.

Available Event widgets

Widget Description
Event details View and edit basic event details.
Incident notifications View a list of related incident templates, add new ones, and associated existing incidents. Launch incident notifications.
Documents View associated documents and add new ones to the event.
Root cause View and add the root cause of the critical event, if applicable.
Notes

Add notes to the dashboard, including up to 4 attachments (for example, images, videos, or documents). Notes are shared across the event. Users can edit or delete their own notes, while Administrators can edit or delete those left by others.

  1. Enter a note of up to 5000 characters in the text field.
  2. Click the upload icon to add up to 4 files.
  3. Once you're done, click Add Note.
Tasks Needing Attention See a list of the tasks that need attention. Click a task to see its details and change the status of the task. You can also add comments to a task, including up to maximum 4 files.
Dashboard Comments

Add comments about the dashboard. Dashboard comments are only shared with others who have access to the dashboard. Users can edit or delete their own comments, while Administrators can edit or delete those left by others.
Task and Note Documents View documents attached to tasks or notes.
Meeting Minutes View a record of what happened during associated meetings.
Audit Log View and export the event audit log or dashboard audit log.

Save dashboards to the event template

You can save the dashboards you configured to the template associated with the critical event. The helps you launch future events with all the components your team and your contacts find most useful.

To save dashboards to an event template:

  1. Open the critical event you want to edit the dashboard for (for example, clicking on the event in the Events dashboard or in the Events list view), make sure you're on the event Home tab then click Configure > Save Dashboards to Event Templates.
  2. Select the checkboxes of the dashboards you want automatically included whenever the associated event template is launched.
    • To include customizations you made to the default dashboard, make sure to select Dashboard.
  3. Click Save.

Going forward, whenever an event that uses the same template is launched, it will include the dashboards you selected.

To confirm that the selected dashboards are associated with the critical event template, you can check it during launch.

  1. From the Critical Events tab, click Launch Critical Event.
  2. Select the critical event template you customized the dashboard for.
  3. Select the Dashboards View checkbox. The dashboard names are displayed from the View link.
