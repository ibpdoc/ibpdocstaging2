Critical Event Management (CEM)

Critical events happen every day. What's critical for your organization can vary widely, from severe weather to workplace violence, from IT outages to attacks on your network. Critical event management lets you decide what threatens your organization's safety, reputation, supply chains, and IT operations.

The Critical Event Management component of Everbridge 360™ provides a suite of solutions to handle your critical events throughout their lifecycle:

  • Access the threats
  • Locate employees
  • Act on the plan
  • Analyze the results

Critical Events Templates

Critical event templates allow you to plan your critical event communications, tasks, and documents before an event occurs. You can find all of your organization's critical event templates on the Critical Events Templates tab. From here, you can create, edit, or copy critical event templates. See Creating a Critical Event Template for more details.

Just like the events themselves, templates can also be organized into folders so you can group similar templates and find them more easily. For example, you could group all your severe weather templates into one folder.

Task List Templates

Task list templates let you create lists of related tasks that need to be completed during an incident. On the Task List Template tab, you can create, edit, or copy these task list templates. See Task List Templates for more details.

Select one or more templates to enable the Actions button. The available actions are:

  • Download
  • Share with Mobile Contacts
  • Remove from Sharing
  • Delete
  • Move to Folder

Widget Library

Administrators can create custom widgets (text and webpage) to add things such as announcements, incident command team structure, or reference materials.

For details on creating custom widgets, see Dashboard Widgets.

If you create a custom widget from the Critical Events dashboard Manage Dashboard screen, it's automatically added to the dashboard (and you can select to add it to the library). If you add a custom widget from the Widget Library screen, you'll need to add it to the dashboard.

Forms

The Forms section may be labelled something different (often "Requests") since it can be renamed by your organization. It's where you find both the Submissions sent using forms and the Form Library.

The Submissions tab shows all of the submissions for your organization and allows you to create a new one. Select one or more submissions to enable the Actions button. The available actions are:

  • Escalate to Critical Events
  • Delete
  • Move to Folder

The Forms Library tab houses all of your organization's forms. Click the Public Link icon to get an externally shareable link you can use to collect data from outside sources.

**image**

To learn more about using forms and managing submissions, see the Forms Module section in the Everbridge Suite help.

Documents

Find and add documents important to your organization on the Documents tab. See Documents for more information on managing documents.

Reports

You can find, create, and download reports for critical events on the Reports tab. Click one from the list to view it and download a PDF. To remove a report, select the checkbox and click Delete.

**image**

For more information on reports, see **Critical Event Report**.

Audit Log

The Audit Log provides a detailed log of a specific critical event, which you can then export as a .csv file.

**image**

For more information on the Audit Log, refer to **Viewing the Event Audit Log**.
