Release 6.7.0.0 Private Cloud

Infinite Blue®Platform Version 6.7.0.0 Release Notes

Release Date

May 2023

Fixed Issues

Following is the list of fixed issues with this release:

Support ID

Bug ID

Public Description
CUS-6421, CUS-9273, PTSP-3032

PA-1896

Integer fields in v6.2.1 contains a value of 0 (zero), the value will display as blank in list views across the system. These fields now display 0.
PTSP-4293

PA-2274

The use of Tomcat 9.0.73 + openjdk 17.0 has been verified.
PTSP-4206, PTSP-4342

PA-2622

When a field has a "Required" attribute, but the import spreadsheet doesn't have a value for that column / field, the import log created showed

Import error: {0}Field "Quantity" must have a value

It was not possible to trace which row in the import for large imports. Rows have been added.

 

PA-3510

When installing an application to a new tenant, create and view formula conditions would merge into one line.
PTSP-4314, CUS-9826

PA-2834

If the administrator who created a batch job has their role changed to no access. The job fails to run. Batch jobs will now always run as the local admin user.

 

PA-2401

When configuring a grid for an object type, only the default object type was visible. Now all types are visible.

 

PA-1379

Any integration codes used for non-object/non-field components should now be optional not required.
