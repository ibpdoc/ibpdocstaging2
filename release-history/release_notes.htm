Enhancement

Following is the enhancement with this release:

Partial Lock

Partial lock preferences is now introduced as a beta feature, where the users can now configure components using partial lock preferences. This feature provides options for Unlocked, Partially Locked, or Locked status for supported components, allowing for better security and confidentiality of sensitive code. However, please note that these enhancements are in the developmental stage and are not recommended for production use.

To access these settings, navigate to the More Actions menu on the Application Setup page and select Partial Lock Preferences. Remember to test these features in a controlled environment before implementing them.

For more information, see Partial Lock Preferences.
