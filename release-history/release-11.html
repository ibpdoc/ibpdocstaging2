Release 5.1 Private Cloud

Infinite Blue® Platform Version 5.1 Release Notes

Release Date

Feb 23rd, 2018

Backward Incompatibility

Following is a change introduced with this release. In case of a custom trigger development to update records, use the DataObject.getFullFieldMap() method, instead of DataObject.getShortFieldMap(), to fetch the current record values
