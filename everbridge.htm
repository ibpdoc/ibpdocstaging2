High level on the latest features

Here's an overall blurb about what's new across the platform. All enhancements are free to existing customers and won't affect preexisting settings, unless otherwise noted. Release Date: October 22 (EU), October 23 (US), unless otherwise noted.

  • And yet more - aren't we productive
  • Advanced Search for Users Without a Role or SSO ID: Administrators can now use Advanced Search to quickly identify users without a role or SSO ID so that they can be promptly corrected. This won't affect preexisting settings.
  • Custom Roles Permissions Grid on Roles Page: The Custom Roles Permissions Grid has been moved from the Support Center to the Roles page for ease of access. You can find it in the dropdown menu next to the legacy Permissions Grid.
  • Improved Layout for Manager Portal SSO Settings Page: Administrators will now see an improved interface for the Single Sign-On for Manager Portal page. The new, modern look uses the same style as other product pages. Additionally, SHA-1, as a deprecated technology, is no longer available as an option for the Secure Hash Algorithm.
