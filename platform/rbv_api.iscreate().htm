rbv_api.isCreate()

Purpose

Returns true if this trigger is called while creating a new record.

Note: This method returns false if called outside of a trigger

Syntax

rbv_api.isCreate()

Parameters

None

N/A

Return value

true or false

Example

if (rbv_api.isCreate()) rbv_api.print("Creating a new record");
