Currency formats

Every currency field and every formula field that returns a value of type Currency allows the selection of a currency format to use (this field-specific format is used for all users) as shown below:

Note: If your object supports the Multi-Currency attribute, you can create a base currency field that corresponds to each currency field to automatically calculate that field's value in the base foreign currency. You can also select the base currency format. See Multi-currency support.
